# Pro Tasker – Full-Stack MERN App
Description
Pro Tasker is a full-stack MERN application that allows users to manage projects and tasks. Users can register, log in, and securely create, update, and delete projects and tasks.
Each project can contain multiple tasks, and each task has a title, description, and status (To Do, In Progress, Done).

***Tech Stack***
**Frontend**
React (Vite)
Axios
React Router
CSS
**Backend**
Node.js
Express
MongoDB (Mongoose)
JWT Authentication
bcrypt
**Features**
User authentication (register & login)
Protected routes
Create, read, update, delete projects
Create, update, delete tasks
Task status management
Persistent login using localStorage
# How to Run Locally
1. **Clone the repository**
git clone:  https://github.com/Sntungane/Full-Stack-MERN-Application

**2. Backend Setup**
cd backend
npm install
Create a .env file:
PORT=5050
MONGO_URI=______________
JWT_SECRET=_______________
Run backend:
npm run dev
**3. Frontend Setup**
cd frontend
npm install
npm run dev
Frontend runs on:
http://localhost:5173
Backend runs on:
http://localhost:5050

**API Endpoints**
Auth
POST /api/auth/register
POST /api/auth/login
Projects (Protected)
GET /api/projects
POST /api/projects
GET /api/projects/
PUT /api/projects/
DELETE /api/projects/
Tasks (Protected)
GET /api/projects//tasks
POST /api/projects//tasks
PUT /api/projects//tasks/
DELETE /api/projects//tasks/

**Authentication**
Uses JWT for authentication
Token is stored in localStorage
Axios attaches token to protected requests
